A script for managing system date and time.

##Requires
<ul>
<li>ntp (optional)</li>
</ul>

##Installation
Install it as:

sudo make install

For Manjaro Linux users its available in the Manjaro repositories.

For Arch Linux users check out http://aur.archlinux.org/packages/timeset/

##Usage
Run the script as root.

 # timeset
